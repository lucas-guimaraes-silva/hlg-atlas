# Noticias do Mercado

Area semanal para conectar os conceitos da trilha com acontecimentos reais.

## Modelo de atualizacao

Para cada noticia:

1. O que aconteceu?
2. Que conceito da trilha aparece aqui?
3. Quem ganha e quem perde?
4. O que muda para o investidor?
5. Que pergunta isso gera para discussao?
