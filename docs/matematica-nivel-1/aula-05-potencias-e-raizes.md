# Aula 5: Potências e raízes

## Comece aqui

Juros compostos funcionam com potências. Sem entender o que é 1,10³ você não consegue calcular quanto R$1.000 vira em 3 anos a 10% ao ano. Esta aula destrava isso.

## Ideia central

Potência é multiplicação repetida. 1,10³ significa 1,10 × 1,10 × 1,10. Raiz é a operação inversa: dado o resultado, qual era a base? Em finanças, potências aparecem nos juros compostos e raízes aparecem quando você quer descobrir a taxa a partir do montante.

## Conceitos essenciais

| Conceito | Significado simples |
|----------|---------------------|
| Base | O número que será multiplicado por si mesmo |
| Expoente | Quantas vezes a base é multiplicada |
| Potência | O resultado: base^expoente |
| Raiz quadrada (√) | Qual número multiplicado por si mesmo dá X? |
| Raiz enésima (ⁿ√) | Generalização — qual base elevada a n dá X? |
| Potência fracionária | x^(1/n) = ⁿ√x — raiz como potência |

## Potências que aparecem em finanças

$$
(1 + i)^n
$$

Esta é a fórmula dos juros compostos. O $(1+i)$ é a base — o fator de crescimento por período. O $n$ é o expoente — o número de períodos. Calcular potências é calcular esse crescimento.

## Raízes em finanças

Se você sabe que R$1.000 virou R$1.331 em 3 anos, qual foi a taxa anual?

$$
taxa = \sqrt[3]{1{,}331} - 1 = 1{,}10 - 1 = 10\%\ ao\ ano
$$

Raiz cúbica do fator de crescimento, menos 1, dá a taxa por período.

## Resumo

- Potência é multiplicação repetida: $a^n = a \times a \times \ldots \times a$ ($n$ vezes).
- Raiz é a operação inversa da potência.
- A fórmula dos juros compostos $(1+i)^n$ usa potência diretamente.
- A próxima aula ensina notação científica — pra ler números grandes sem se perder.
